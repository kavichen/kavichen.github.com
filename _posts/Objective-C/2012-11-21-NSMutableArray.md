## Create a NSMutableArray  
</br>  
  
### `arrayWithCapacity:`  
Creates and returns an `NSMutableArray` object with enough allocated memory to initially hold a given number of objects.  
  
	+ (id)arrayWithCapacity:(NSUInteger)numItems
