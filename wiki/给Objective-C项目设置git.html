创建git的步骤

  1. 打开Terminal
  2. cd /Developer/SULIMN/进入项目文件夹
  3. find .列出所有文件
  4. 找到以.xcuserstate结尾的文件,并复制
  5. 输入以下命令,忽略此条文件更新信息 
    


  6. 
输入git init


  7. 
输入git add .


  8. 
输入git commit -m "xxxxxx"


  9. 
输入git push






在Xcode中的git操作


查看历史版本


    

  1. 
打开Xcode中该项目


  2. 
点击右上角Organizer


  3. 
选择Repositories


  4. 
找到项目提交版本旁边的三角,并打开


  5. 
单击View Changes
