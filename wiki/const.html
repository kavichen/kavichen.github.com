const

  • const char ch = 'a' - 字符常量
  • const int a[5] = {1,2,3,4,5}; - 数组常量
  • const int *p = a - p是指向常量的指针
    • 不可以通过给指针复制来改变数组，例如：
    • *p=10是错误的
  • int *const p = a - p是指针常量
    • 不可以改变指针的变量，例如：
    • *p = 2是允许的
    • p++是不允许的
  • const int * const p = a - p是指向常量的指针常量
    • 即不允许通过指针来改变数组，又不允许改变指针的变量
