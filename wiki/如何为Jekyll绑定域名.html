如何为Jekyll绑定域名

之前从来没有帮顶过域名，第一次用，在google简单搜索了一下，居然成功了。

我这里用的是 http://paomi.com 泡米网的域名管理服务。如果选用其他域名地址，原理也是一样的。

步骤

#在本地代码库里新建名为CNAME文本文件，在Terminal中输入，以为例:example.com 

  echo 'example.com' > CNAME

然后push 

  git add CNAME
  git commit -am "CNAME ADDED"
  git push

#登陆域名管理，在域名解析中添加

-主机记录：@

-记录类型：A

-路线类型：默认

-记录值：204.232.175.78

-其他：默认
