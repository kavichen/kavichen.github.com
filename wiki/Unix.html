  1. [Tab]補全
  2. [Ctrl]-c 停止當前程序
  3. [Ctrl]-d 代表按鍵輸入exit

Linux系統的在線求助man page與info page #man page

  • NAME 簡短的命令.數據名稱說明
  • SYNOPSIS 簡短的命令執行語法(syntax)簡介
  • DESCRIPTION 較為完成的說明,這部份最好仔細看
  • OPTIONS 針對SYNOPSIS部份中,有列舉的所有可能的選項說明
  • COMMANDS 當這個程序或者軟件執行的時候,可以在此程序或軟件中中執行的命令
  • FILES 這個程序貨數據所是用或者參考或者連接到的某些文件
  • SEE ALSO 這個命令貨數據有相關的其他說明
  • EXAMPLE 一些可以參考的範例
  • BUGS 是否有相關的錯誤
  • 按下/date就可以超找文中的date關鍵字 #info page

正確的關機方法

  1. 需要關機時,要注意如下:
    • 用who命令查看目前有誰在線
    • 用netstat -a網絡聯機狀態
    • 用ps -aux查看後臺程序
    • 用shutdown命令來關機
  2. 慣用關機命令shutdown
