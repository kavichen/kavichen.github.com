Mac連接RPi步驟

RPi端

  1. sudo apt-get install tightvncserver
  2. tightvncserver
  3. To address a virtual screen ,you need two things: the IP address of the Pi and the port of the screen. VNC's base port is 5900, so to access screen number 1, you have to use port 5901.

Mac端

  1. Open safari
  2. Type network address vnc://xx.xx.xx.xx:5901
  3. Enter the password

RPi連接Mac步驟

Mac端

  1. System Performence => Sharing => Screen Sharing

RPi端

  1. Install sudo apt-get install xtightvncviewer
  2. xtightvncviewer 10.0.10.45:5900
