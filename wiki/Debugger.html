Debugger

Exception Breakpoints

在编辑每个程序时，都要设置Exception breakpoint.

设置方法

  1. 按Command + 6
  2. 找到左下角“+”
  3. Add Exception breakpoint ...

作用

在当前exception的时候停下程序

调用Debugger

设置方法

  1. 在问题语句上设置断点】
  2. 用step into调试
  3. 需要观察参数时，在Console中输入命令
  4. (gdb)print [...], 里面不能使用.,要用例如[self operandStack] / po [self operandStack](po = print object)

在断点上设置条件

  1. 右键点击断点
  2. 选择Edit Breakpoint
  3. 在Condition框中输入条件，例如: resutl == 0
  4. 点击Done

在类中建立自己的description

  1. 在类中建立description方法，例如： 
        - (NSString *)description
      {
        return [NSString stringWithFormat:@"stack = %@",self.operandStack];
      }
  1. 在Console的(gdb)中输入po self
