第一步 - 远程登陆

  1. ssh root@12.34.56.78
  2. 登陆vps

第二步 - 修改密码

  1. passwd

第三步骤 - 新建新用户

  1. adduser kavi

第四步 - 分配管理员权限

  • 为了阻止操作者用管理员权限进行误操作
  • 用sudo执行的命令会被记录在/var/log/secure中
  1. 执行visudo，Ubuntu会用nano打开sudo configuration
  2. 修改文件如下： 
      # User privilege specification
  root  ALL=(ALL:ALL) ALL
  kavi  ALL=(ALL:ALL) ALL

第五步 - 配置ssh

  1. 打开配置文件sudo vi /etc/ssh/sshd_config
  2. {{{ Port 22 Protocal 2 PermitRootLogin no AllowUsers kavi }}}
    • Port为登陆接口
    • PremitRootLogin:是否可以用root登陆
    • AllowUsers:允许登陆的User

第六步 - 重载ssh

  1. relaod ssh
  2. 登陆ssh -p 22 kavi@12.34.56.78
