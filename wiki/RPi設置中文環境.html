步驟

  1. sudo apt-get update
  2. 安裝中文字型，文泉驛微米黑、文泉驛正黑、文泉驛點陣宋體。
    • sudo apt-get install tff-wqy-microhei tff-wqy-zenhei xfonts-wqy
  3. 然後將locale設定成繁體中文
    • sudo dpkg-reconfigure locales
  4. 選擇需要的中文編碼
    • zh_CN GB2312
    • zh_CN.GB18030 GB18030
    • zh_CN.GBK GBK
    • zh_CN.UTF-8 UTF-8
  5. 按回車確認
  6. 默認選擇 zh_CN.UTF-8
  7. 完成
