Supervisor

配置文件

  1. sudo vi /etc/supervisor/supervisord.conf
  2. 增加需要进程管理的 python 程序 
        [program:hello] // 程序名称
    command=python /var/www/hello.py // 程序命令行
    autostart=true // 自动开始运行
    stdout_logfile=/home/kavi/log/wx_log/ //捕获标准输出

基本命令

  • sudo supervisor status hello - 查看 hello 程序进程情况
  • sudo supervisorctl start hello - 启动 hello 服务
  • sudo supervisorctl stop hello - 停止 hello 服务
